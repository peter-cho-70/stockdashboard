## 1. 제품 개요 (Product Overview)

### 1.1 제품명
**StockMind** — AI 기반 개인 주식 인텔리전스 플랫폼

### 1.2 제품 비전
유튜브 경제방송, 뉴스, 시세 데이터를 AI가 통합 분석하여 개인 투자자가 **내 포트폴리오 중심으로** 시장 흐름을 빠르게 파악하고, 종목별·섹터별 이슈를 시계열로 추적할 수 있는 개인 맞춤형 주식 인텔리전스 플랫폼.

### 1.3 핵심 차별점
| 일반 주식 앱 | StockMind |
|------------|-----------|
| 시세 조회 중심 | 내 종목 중심 AI 분석 |
| 뉴스 나열 | 섹터·종목별 이슈 트래킹 |
| 유튜브 따로 봄 | 유튜브 채널 AI 요약 통합 |
| 정보 파편화 | 과거 이슈와 현재 연결 |

### 1.4 타깃 사용자
- 국내·해외 주식을 함께 보유한 개인 투자자
- 경제 유튜브를 자주 시청하는 투자자
- 섹터 로테이션, 매크로 이슈에 관심 있는 투자자

## 2. 시장 조사 및 경쟁 분석

### 2.1 주요 유사 서비스

|--------|------|------|
| 씽크풀 | AI 매매신호, 종목 분석 | 유튜브 분석 없음, 유료 |

## Module 1: 포트폴리오 트래커

### 3.1 종목 관리
**FR-101** 종목 등록 및 관리
- 국내 주식 (KOSPI/KOSDAQ), 미국 주식, ETF 등록
- 보유수량, 평균매입단가, 매입일 입력
- 종목별 메모 및 투자 thesis 기록 가능

**FR-102** 자동 시세 갱신 (핵심 기능)
- 국내 장: pykrx + 공공데이터포털 API 활용
- 미국 장: yfinance API 활용
- 갱신 시점 (자동 스케줄링):

| 시점 | 시간(KST) | 대상 |
|------|-----------|------|
| 국내 장 시작 전 | 08:50 | 전일 해외 마감가, 시황 요약 |
| 국내 장 마감 후 | 15:35 | 국내 종가 확정 갱신 |
| 미국 장 시작 | 23:35 | 미국 장 오픈 시황 |
| 미국 장 마감 | 07:05 | 미국 종가 확정 갱신 |

### 3.2 포트폴리오 대시보드
**FR-103** 수익 현황 요약
- 종목별 현재가 / 평균단가 / 수익률(%) / 수익금액
- 전일 대비 등락률 및 등락금액
- 전체 포트폴리오 총 평가금액, 총 수익률

**FR-104** 알림 기능 (5% 룰)
- 당일 ±5% 이상 변동 종목 → 🔴🟢 강조 표시 + 알림
- 알림 채널: 웹 푸시 / 이메일 / (선택) 카카오톡
- 임계값 사용자 커스터마이징 가능 (기본 5%)

**FR-105** 변동 히스토리 뷰
- 날짜별 포트폴리오 총 수익률 추이 (일별 라인 차트)
- 종목별 날짜별 가격 변동 테이블

### 3.3 이슈 노티 및 정보 갱신
**FR-106** 이슈 감지 알림
- AI가 해당 종목의 뉴스/이슈 감지 시 알림
- 알림 클릭 → 해당 종목 상세 페이지로 이동
- 사용자가 직접 메모/정보 갱신 가능

## Module 2: 인텔리전스 허브

### 3.4 콘텐츠 수집 및 분석 엔진

**FR-201** 유튜브 채널 등록 및 관리
- 경제/주식 유튜브 채널 URL 등록
- 신규 영상 자동 감지 (YouTube Data API v3)
- 영상별 자동 분석 큐 등록

**FR-202** 유튜브 영상 AI 분석 (Gemini API)
- YouTube URL → Gemini API 직접 분석
- 추출 항목:
  - 언급된 종목명/티커
  - 언급된 섹터
  - 핵심 투자 thesis (불릿 3~5개)
  - 긍정/부정/중립 톤 분석
  - 주요 경제 이슈 키워드
  - 타임스탬프별 주요 발언

**FR-203** 다중 입력 소스 지원
- YouTube 링크 붙여넣기 → AI 분석
- 뉴스 기사 URL 링크 → 본문 크롤링 후 AI 분석
- 텍스트 직접 입력 → AI 분석
- 분석 결과 모두 동일한 DB 구조로 저장

### 3.5 인텔리전스 대시보드

**FR-204** 메인 인텔리전스 화면
- **오늘의 주요 이슈** (Daily Brief): 오전/오후 자동 생성
- **내 종목 이슈 피드**: 보유 종목 관련 최신 언급 타임라인
- **Hot 섹터 트래커**: 최근 7일 언급 빈도 상위 섹터
- **주요 경제 일정 캘린더**: 연준 회의, 실적발표, 경제지표 발표일

**FR-205** 섹터별 이슈 트래킹
- 섹터 카테고리: 반도체, AI·빅테크, 2차전지, 바이오·헬스케어, 금융, 에너지, 소비재, 자동차, 방산, 부동산·리츠
- 섹터별 페이지: 최신 이슈, 언급 종목, 섹터 주가 지수 차트
- 섹터 간 로테이션 흐름 시각화

**FR-206** 종목별 이슈 트래킹
- 종목별 이슈 타임라인 (날짜 + 출처 + 요약)
- 과거 이슈와 주가 변동 오버레이 차트
- "이 종목이 언급된 영상 모두 보기" 기능

**FR-207** 월별/일별 이슈 아카이브
- 캘린더 뷰: 날짜 클릭 → 해당일 분석 내용 전체 조회
- 월별 요약: 그달 가장 핫했던 섹터, 종목, 키워드 정리
- 검색: 키워드, 종목명, 날짜 범위로 과거 이슈 검색

**FR-208** 미국·해외 주식 이슈
- S&P500, NASDAQ 주요 종목 이슈
- 매크로 이슈 (금리, 환율, 원자재)
- 해외 주요 경제 뉴스 AI 요약

### 3.6 즐겨찾기 카테고리
**FR-209** 사용자 커스텀 카테고리
- "내 관심 섹터", "내 관심 종목" 별도 탭으로 관리
- 관심 키워드 등록 → 해당 키워드 언급 시 우선 노출
- 카테고리별 주간 브리핑 리포트 자동 생성

## Module 3: 차트 AI 분석기

### 3.7 주가 차트 분석

**FR-301** 인터랙티브 차트
- 국내/미국 주식 캔들차트 (일봉/주봉/월봉)
- 보조지표: MA(5/20/60/120), 볼린저밴드, MACD, RSI, 거래량
- 보유 종목 포트폴리오 내에서 빠른 전환

**FR-302** AI 차트 패턴 분석 (Gemini)
- 차트 이미지 → Gemini Vision 분석
- 감지 패턴: 헤드앤숄더, 이중바닥/천장, 삼각수렴, 골든크로스 등
- 분석 결과: "현재 차트가 보여주는 특징 3가지" + 주의사항

**FR-303** 향후 변동 예측 (AI 기반)
- 기술적 지표 + 최근 이슈 + 시장 센티멘트 종합 분석
- "향후 1~4주 관전 포인트" AI 코멘트 생성
- ⚠️ 면책 조항 필수 표시: "본 분석은 참고용이며 투자 결정의 근거가 될 수 없습니다"

**FR-304** 이슈-차트 연결 뷰
- 차트에 주요 이슈 발생일 마커 표시
- 마커 클릭 → 해당 날짜 이슈 팝업
- "이슈가 주가에 미친 영향" 자동 코멘트

## 4. 비기능 요구사항 (Non-Functional Requirements)

### 4.1 성능
- 대시보드 초기 로딩: 3초 이내
- 시세 갱신: 갱신 예약 시간 기준 5분 이내 완료
- AI 분석 (영상 1개): 60초 이내 완료

### 4.2 데이터 저장
- 로컬 SQLite (개인용) 또는 Supabase (멀티기기)
- 분석 이력 최소 2년 보존
- JSON 백업/복원 기능

### 4.3 보안
- API 키 로컬 암호화 저장
- 개인 포트폴리오 데이터 외부 전송 없음 (AI 분석 시 종목명만 전송)

### 4.4 알림
- 웹 푸시 알림 (PWA)
- 이메일 알림 (선택)
- 카카오톡 알림 (선택, 카카오 알림톡 API)

## 5. 기술 아키텍처

### 5.1 전체 구조
```
[Frontend]
Next.js 14 (App Router) + TailwindCSS + Recharts/TradingView

[Backend]
FastAPI (Python) — API 서버
APScheduler — 자동 갱신 스케줄러

[Data Layer]
SQLite / Supabase (PostgreSQL)

[외부 API]
├── pykrx — 국내 주식 시세
├── yfinance — 미국/해외 주식 시세
├── YouTube Data API v3 — 채널 영상 목록
├── Gemini API — 영상/텍스트 분석
├── FinBERT — 뉴스 감성 분석
└── 공공데이터포털 — 금융위 주식시세

[알림]
Web Push (PWA) / SMTP / 카카오 알림톡
```

### 5.2 스케줄러 구조
```python
# 자동 갱신 스케줄 (KST 기준)
08:50 — 전일 미국 마감가 갱신 + 오전 브리핑 AI 생성
15:35 — 국내 종가 갱신 + 5% 룰 알림 체크
23:35 — 미국 장 시작 시황 요약
07:05 — 미국 장 마감가 갱신 + 익일 주요 일정
```

### 5.3 AI 분석 파이프라인
```
YouTube URL 입력
    ↓
YouTube Data API → 영상 메타데이터
    ↓
Gemini API (YouTube URL 직접 분석)
    ↓
구조화된 JSON 추출
{종목[], 섹터[], 키워드[], thesis[], 톤}
    ↓
DB 저장 (종목·섹터 매핑 포함)
    ↓
내 보유 종목 매칭 → 알림 트리거
```

## 8. 주요 고려사항 및 제약

### 8.1 데이터 제약
- KRX 실시간 시세: 개인 참고용 가능, 상업적 재배포 불가
- Gemini YouTube 분석: 공개 영상만 가능, 1일 8시간 분량 제한
- yfinance: 무료이나 안정성 제한, 상업용 시 유료 API 검토 필요

### 8.2 법적 고려
- AI 투자 예측 결과에 면책 조항 필수
- 뉴스 기사 크롤링 시 저작권 준수 (요약만 저장, 원문 링크)
- 개인정보 처리방침 (포트폴리오 데이터 처리)

### 8.3 기술 리스크
| 리스크 | 완화 방안 |
|--------|---------|
| Gemini API 비용 증가 | 분석 요청 큐잉, 캐싱, 중복 분석 방지 |
| pykrx 서버 차단 | 요청 간격 조절, 공식 API 대체 |
| YouTube 자막 없는 영상 | Whisper STT 대체 분석 |
| 시세 갱신 실패 | 재시도 로직, 실패 알림 |

## 9. 성공 지표 (Success Metrics)

| 지표 | 목표 |
|------|------|
